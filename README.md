# 🛰️ EuroSAT Land Cover Classification

## 📌 Overview

This project uses deep learning to classify satellite images into 10 land-cover categories using the EuroSAT dataset.

## 🚀 Features

* EfficientNet-B0 (Transfer Learning)
* PyTorch-based training pipeline
* Gradio-based interactive web app
* Google Colab compatible

## 📊 Dataset

EuroSAT dataset (Sentinel-2 satellite images)

Classes:

* AnnualCrop
* Forest
* HerbaceousVegetation
* Highway
* Industrial
* Pasture
* PermanentCrop
* Residential
* River
* SeaLake
